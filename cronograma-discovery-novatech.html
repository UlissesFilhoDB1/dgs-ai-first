Dependências críticas

  • Acessos → tudo: a fase de Intent só inicia com leitura confirmada nas 3 fontes — é o caminho crítico.
  • Dossiê → entrevistas: nenhuma entrevista antes do Dossiê; o roteiro nasce dos achados dos agentes (temas, gaps, contradições).
  • H3 → H6: o golden dataset só é construído após a adjudicação — validar respostas contra documentos contraditórios gera gabarito contraditório.
  • Dono de governança nomeado → go/no-go: condição para encerrar o discovery com decisão.

NovaTech fornece — quando

  • Antes do D1: acessos de leitura + Azure provisionado + lista de stakeholders.
  • D1-2: ponto focal de TI + export do sistema de chamados.
  • D4-8: 1h por área (review) · agenda de 6-8 entrevistas de 45min · meio período dos decisores com mandato (D7-8).
  • D8-9: dono da governança nomeado + 2-3 especialistas para o golden dataset.
  • D10: 1h da diretoria para o go/no-go.
